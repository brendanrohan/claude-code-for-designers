# Weather Widget — PRD

**What is this?**
A weather widget that shows current conditions with illustrated backgrounds that change based on weather. Built as an experiment to demonstrate what designers can build with Claude Code.

**Who is it for?**
Designers and makers curious about building with AI-assisted coding tools.

**What it does:**
- Detects user's current location automatically
- Shows current temperature in Fahrenheit
- Displays time until sunrise or sunset (whichever is next)
- Background illustration changes based on weather conditions (sunny, cloudy, rainy, snowy, etc.) — same moody, layered silhouette style as the Lofoten design
- Shows current date/time and location name

**What it doesn't do:**
- No forecast (hourly or daily)
- No Celsius toggle
- No manual location picker
- No additional weather details (humidity, wind, etc.)
